### Installation

_Below is an example of how you can instruct your audience on installing and setting up your app. This template doesn't rely on any external dependencies or services._

1. Clone the repo
   ```sh
   git clone https://github.com/hebe889900/uberclone.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Request Google Map API KEY:
   1. Go to google api console at https://console.developers.google.com/apis
   2. Click Enable Apis and service
   3. Enable the following:
        Directions API
        Places API
        Distance Matrix API
   4. Go to credentials and copy the API Key then in the .env file paste it at GOOGLE_MAPS_APIKEY.
   5. Enable billing for your project, otherwise it may not work.
4. Start the application
   ```sh
   expo start
   ```
